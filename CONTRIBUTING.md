# Contributing

Thanks for contributing to Hood College Signal Atlas.

## Current Project Stage

This repository is in the planning and bootstrap stage. Before opening implementation-heavy pull requests, please make sure the proposed change aligns with the planning documents in `docs/`.

## Contribution Guidelines

1. Prefer small, reviewable pull requests.
2. Keep Hood College disambiguation strict. This project only covers Hood College in Frederick, Maryland.
3. Preserve article-level evidence for any extracted people or connections.
4. Avoid adding broad or noisy sources without documenting why they are trustworthy.
5. Update documentation when changing behavior, schema, sources, or output format.

## Development Expectations

- Document new commands in `README.md`.
- Keep generated data out of commits unless the workflow is explicitly designed to version it.
- When changing extraction logic, note whether reprocessing is required.

## Pull Request Checklist

- The change matches the current project scope.
- Documentation stays in sync with the code.
- New sources or heuristics include rationale.
- Any known risks or follow-up work are called out in the PR description.
